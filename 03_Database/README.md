# 데이터베이스 교육 — 웹 개발의 데이터 기초

HTML/CSS/JS로 화면을 만들 줄 알게 된 여러분이, 이제 **데이터를 저장하고 다루는 법**을 배웁니다.
이 모듈은 [스프링부트 교육](../Spring/SpringBoot/README.md)의 **선수 과목**입니다 —
여기서 SQL과 JDBC로 직접 해 본 일들을, 스프링이 어떻게 대신해 주는지 아는 사람과 모르는 사람의 이해 깊이는 완전히 다릅니다.

![H2 콘솔에서 SQL 실습](./images/db_02_h2쿼리결과.png)

- **실습 도메인은 스프링 교육과 같은 "자기소개서(intro 테이블)"** 입니다. 여기서 만든 테이블 구조가 스프링 실습에서 다시 등장합니다.
- 05는 Docker Desktop 설치가 필요한 선택 과정입니다. 환경이 안 되면 건너뛰어도 됩니다.

## 시작 전 준비물

1. **JDK 17 이상** — `java -version`으로 확인 (스프링 교육과 동일)
2. **H2 데이터베이스 jar** — 별도 설치 없이 jar 파일 하나면 됩니다. 받는 법은 [01 문서](./01_데이터베이스_기본개념.md)에서.
   (이미 [샘플/jdbc/lib](./샘플/jdbc/lib/)에도 들어 있습니다)
3. 텍스트 편집기 또는 IDE — 04의 자바 실습은 메모장+터미널로도 가능합니다.

## 샘플 코드

[샘플/jdbc](./샘플/jdbc/)에 04 문서의 단계별 완성 코드 6개가 들어 있습니다. 실행 방법은 해당 폴더의 README와 04 문서 참고.

## 이 모듈이 끝나면

- 테이블을 설계(CREATE)하고 CRUD SQL을 자유롭게 쓸 수 있습니다.
- 자바 프로그램에서 DB에 접속해 데이터를 저장/조회할 수 있고, SQL Injection이 왜 위험한지 압니다.
- 접속 정보와 권한이 무엇인지 알고, 운영 DB를 대하는 기본 보안 수칙이 몸에 있습니다.
- 다음 단계: [스프링부트 교육](../Spring/SpringBoot/README.md) — 오늘 직접 한 일을 스프링이 대신해 주는 것을 확인하세요.
