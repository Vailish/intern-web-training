## 시작 전 준비물

1. **JDK 17 이상** — 터미널에서 `java -version`으로 확인
2. **IDE** — IntelliJ IDEA Community(무료) 권장. VSCode(+Java/Spring 확장), 이클립스(STS)도 가능
3. **자바 기초 문법** — 클래스/객체, 메서드, 인터페이스 개념. 부족하면 [점프 투 자바](https://wikidocs.net/book/31)로 병행 학습
4. **데이터베이스 기초** — [데이터베이스 교육 모듈](../../03_Database/README.md)(SQL, JDBC)을 먼저 완주하기를 권장합니다.
   스프링이 대신해 주는 일이 무엇인지 아는 채로 배우면 이해 깊이가 완전히 달라집니다.

## 완성 샘플 프로젝트

[샘플/intro](./샘플/intro/) 폴더에 **실행 가능한 완성본**이 들어 있습니다.
직접 만들다 막혔을 때 비교용으로 쓰세요. 실행 방법:

```powershell
cd 샘플/intro
.\gradlew.bat bootRun      # macOS/리눅스: ./gradlew bootRun
```

브라우저에서 `http://localhost:8080` 접속. 종료는 콘솔에서 `Ctrl + C`.
(첫 실행은 라이브러리를 내려받느라 몇 분 걸립니다.)

## 이 교육이 끝나면

- 브라우저의 요청이 컨트롤러 → 서비스 → 리포지토리 → DB를 거쳐 화면으로 돌아오는 흐름을 설명할 수 있습니다.
- 폼으로 데이터를 받아 DB에 저장하고, 목록/상세 화면으로 보여주는 기능을 스스로 만들 수 있습니다.
- 이 구조는 회사 실무의 eGovFrame(전자정부 표준프레임워크)과 동일해서, 실무 코드를 열었을 때 "어디에 무엇이 있는지"가 보이게 됩니다.
