🪙 JWT 토큰 방식 로그인 데모

아래 순서대로 눌러 보면 토큰이 어떻게 발급되고 쓰이는지 볼 수 있습니다.

1. 로그인 → 토큰 발급 (POST /api/auth/login)
실습 계정: admin/admin123 (관리자), user/user123 (일반) 
아직 로그인하지 않음
저장된 토큰: (없음)
2. 내 정보 조회 (GET /api/me) — 로그인만 하면 OK

3. 관리자 API (GET /api/admin/dashboard) — ADMIN 권한 필요
