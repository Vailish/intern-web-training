🔁 JWT + 리프레시 토큰 데모

액세스 토큰은 60초 후 만료됩니다. 만료 후 /api/me를 부르면 401이 뜨고, 리프레시하면 다시 됩니다.

액세스 토큰: (없음)
리프레시 토큰: (없음)
1. 로그인 → 토큰 2종 발급 (POST /api/auth/login)
admin/admin123 (관리자), user/user123 (일반) 
아직 로그인하지 않음
2. 보호된 API 호출 (액세스 토큰 사용)
60초 지난 뒤 눌러 보면 액세스 토큰이 만료되어 401이 뜹니다. 
-
3. 액세스 토큰 재발급 (POST /api/auth/refresh)
리프레시 토큰도 새 것으로 회전됩니다. 위의 토큰 값이 바뀌는 것을 보세요. 
-
4. 로그아웃 (POST /api/auth/logout)
폐기 후 다시 리프레시를 누르면 401 이 뜹니다(회전/재사용 감지).
