🧩 Flexbox 완전 정리

이 파일 하나만 봐도 Flexbox의 핵심 속성을 모두 이해할 수 있도록 "코드 + 결과 화면"을 함께 보여줍니다.
Flexbox는 display: flex를 부모(container)에 적용하는 순간부터 시작됩니다.

1. flex-direction — 배치 방향

자식 요소들을 어느 방향으로 나열할지 정합니다.

.container { display: flex; flex-direction: row; }

5. flex-grow — 남는 공간을 나눠 가지는 비율

컨테이너에 남는 공간이 있을 때, 비율대로 더 차지하게 만듭니다.

.box2 { flex-grow: 2; } /* 나머지는 flex-grow: 1 */
flex-grow: 1
flex-grow: 2 (2배 차지)
flex-grow: 1

6. gap — 아이템 사이 간격

margin을 일일이 주지 않고 컨테이너에서 한 번에 간격을 지정합니다.

.container { display: flex; gap: 16px; }
