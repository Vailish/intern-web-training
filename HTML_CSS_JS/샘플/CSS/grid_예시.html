📐 CSS Grid 완전 정리

이 파일 하나만 봐도 Grid의 핵심 속성을 모두 이해할 수 있도록 "코드 + 결과 화면"을 함께 보여줍니다.
Grid는 display: grid를 부모(container)에 적용해 행(row)과 열(column)을 동시에 짜는 2차원 레이아웃 방식입니다.

1. grid-template-columns / rows — 격자 만들기

몇 개의 열(세로줄)과 행(가로줄)을 만들지, 각 칸의 크기를 정합니다.

.container { display: grid; grid-template-columns: 1fr 1fr 1fr; }

2. repeat() — 반복되는 열을 간단하게 표현

1fr 1fr 1fr 1fr 대신 repeat(4, 1fr)로 줄여서 쓸 수 있습니다.

.container { grid-template-columns: repeat(4, 1fr); }

3. fr 단위로 비율 다르게 주기

fr은 "남는 공간을 나누는 비율" 단위입니다. 1fr : 2fr : 1fr이면 가운데가 2배 넓어집니다.

.container { grid-template-columns: 1fr 2fr 1fr; }

4. grid-column / grid-row — 여러 칸 병합하기

특정 요소가 가로 또는 세로로 몇 칸을 차지할지 직접 지정할 수 있습니다.

.box1 { grid-column: span 2; } /* 가로 2칸 차지 */ .box2 { grid-row: span 2; } /* 세로 2칸 차지 */

5. grid-template-areas — 이름으로 레이아웃 그리기

실제 페이지 레이아웃(헤더/사이드바/본문/푸터)을 만들 때 가장 직관적인 방법입니다.

.container { grid-template-areas: "header header" "sidebar main" "footer footer"; } .header { grid-area: header; } .sidebar { grid-area: sidebar; }

7. 실전 예제 — auto-fit + minmax 반응형 카드

미디어쿼리 없이도 화면 크기에 맞춰 칸 수가 자동으로 조절됩니다. (브라우저 창 크기를 줄여보세요)
